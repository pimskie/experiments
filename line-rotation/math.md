## Slope
* slope: vertical rise / horizontal run
* slope: (y2 - y1) / (x2 - x1)

* Standard form: Ax + By = C 

## Perpendicular distance
* point			(m, n)
* line:			Ax + By + C = 0
* distance = 	(Am + Bn + C) / sqrt((A * A) + (B * B))
